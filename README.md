
🚀 IRIS
Communication Without Video
🧠 What if video was never the right way to communicate?

Every call you’ve ever made assumes one thing:

You must transmit pixels to transmit presence.

IRIS challenges that assumption.

⚡ The Idea

IRIS is a real-time communication system that transmits:

Motion
Intent
Timing
Presence

Instead of video frames.

🎯 One-Line Explanation

IRIS doesn’t send what you look like.
It sends what you are doing.

🔍 The Problem with Video
📦 Massive data (MBs/sec)
📉 Breaks under poor network
❄️ Freezes, blurs, collapses
🔁 Sends redundant pixels
💡 The IRIS Insight

Humans don’t communicate in pixels.

We communicate through:

Head movement
Eye direction
Micro-timing
Behavioral cues

👉 IRIS transmits only that layer.

⚙️ How It Works
User → Camera → Intent Extraction → Tiny Data → Receiver → Presence Reconstruction
Sender:
Tracks motion (x, y, time)
Converts behavior → numbers
Sends minimal JSON packets
Receiver:
Reconstructs presence
Renders avatar locally
Infers continuity
